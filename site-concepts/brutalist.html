Features

Direct-site parsing

Reads the target website first. No third-party resolver wins by default.

Apple-touch priority

Prefers high quality icons when sites expose proper mobile assets.

Bulk group processing

Fetch icons for full groups. Concurrent, cancellable, and resumable.

Android URL mapping

Maps packages to web domains, then fetches native-looking icons.

SHA-256 deduplication

Content-addressed storage prevents duplicate icons and saves space.

Privacy toggles

Disable third-party lookups or synthetic fallbacks. You control remote requests.
