Fetch. Map. Maintain. Favicon made easy.

One-click fetch

Download a favicon for the selected entry in seconds.

Bulk group processing

Fetch favicons for every entry in a group with one action.

Android app mapping

Map package names to official web URLs and fetch icons automatically.

Database-wide maintenance

Scan and update missing or outdated favicons across the database.

System proxy support

Uses the system default proxy configuration through the shared HTTP client.

Native integration

Fast KeePass 2.x plugin behavior that feels at home.

How it works

1

Select

Choose an entry, group, or the full database.

2

Fetch

KeeFetch finds the best favicon from the target website first.

3

Save

The icon is embedded into the KeePass database.

4

Repeat

Process many entries in bulk or during scheduled maintenance.

5

Enjoy

A clean visual database that is easier to scan.
